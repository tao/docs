A collection containing all the configured sites as `Statamic\Sites\Site` objects which you can loop over using a tag pair.

``` php
// config/statamic/sites.php

'sites' => [
    'english' => [
        'name' => 'English Site',
        'locale' => 'en_US',
        'url' => 'http://mysite.com/',
    ],
    'french' => [
        'name' => 'French Site',
        'locale' => 'en_FR',
        'url' => 'http://mysite.com.fr/',
    ]
]
```

```
{{ sites }}
    {{ handle }}
    {{ name }}
    {{ locale }}
    {{ short_locale }}
    {{ url }}
    
{{ /sites }}
```

``` output
english
English Site
en_US
en
http://mysite.com/

french
French Site
fr_FR
fr
http://mysite.com.fr/

```
